# TaskTrack - A Task Manager

This is a full-stack application that takes in user data to store tasks and display them to the user

## Demonstration

![](demonstration.gif)

## Built With

  - PHP
  - HTML
  - CSS
  - MySQL

## Features

- Add tasks
- Edit Tasks
- Delete Tasks
- Login/Register
- Logout
- Saves Tasks Based on Username
- Tasks and User Data Saved in Data Base
